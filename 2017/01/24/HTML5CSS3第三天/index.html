HTML5css3基础（三）

发表于 2017-01-24

HTML5CSS3第三天

渐变

  • 给需要的元素设置图片属性（background或者background-image）
  • 渐变在谷歌浏览器中，过度效果不全部支持，也就是说设置了渐变，设置了transition想实现渐变 颜色的变化效果，在谷歌中是不起作用的
  • 但是在ie中支持过度效果，不过在实际开发中不会这样写

渐变的分类

1.0线性渐变（朝着某个方向反向渐变）

  • 说明：径向渐变是反向渐变，设置的红色到白色，显示出来就是白色到红色

  • 语法： background-image: linear-gradient(角度/to 方向,颜色1，颜色2，颜色3…);

  • 第一个参数可以是角度加单位deg,也可以是方向to left ,to right,to top,to bottom,

  • 后面的参数表示颜色，支持%和像素，可以是多个颜色。

  • 颜色1+ 范围 表示该颜色 从0开始 直到 某个值

  • 颜色2+ 范围 表示改颜色 从 某个值 开始

  • 颜色2+ 范围 表示该颜色 到某个值 结束

2.0径向渐变（圆形或者椭圆）

  • 语法： background: radial-gradient(半径1 半径2 at 方向,颜色1 ,颜色2，…);

  • 半径:如果只设置一个就是圆，如果设置两个就表示椭圆，但是如果半径1和半径2一样，那么也是圆

  • 方向：可以是坐标 也是可以具体的方向

background新属性

background-size:（设置背景图片的大小）具体大小 corver contain

说明：可以是具体的大小或者corver或者contain

  • background-size: 300px 300px;
  • background-size: cover: 等比例放大 直到两边都顶格,如果图片由于比例问题,出现超出范围的情况, 会出现图片 缺失的问题。出去的图片会被切掉。
    ​    background-size:  **contain**  等比例放大 某一边顶格 立刻停止放大 
![liyf](/img/liyf.jpg)

    background-origin：（背景图开始的位置）border-box/content-box/padding-box

  • background-origin: border-box; 从边框的位置开始
  • background-origin: content-box; 从padding里面开始
  • background-origin: padding-box; 从padding开始

    background联写

    语法： background: 图片 repeat 位置 /大小 origin
    注意：在css3中支持设置多个背景，但是一般会在 末尾设置一个 背景颜色，当图片加载不出来 或者由于网速 ，需要等待才会出现时，起码让用户看到一个 颜色，提升用户体验。

相框属性

说明：添加相框属性，兼容性较差，实际开发的时候为了保证，用户起码有一个 最基本的相框
会先设置一个 border属性 ， 如果不设置border属性 ，那么就无法使用图片相框，当浏览器不支持 border-image属性时 ，会使用默认的相框来显示

  • border-image-source: url(‘img/border_box.png’); 设置边框的图片地址
  • border-image-slice: 27 27; 设置图片的切割的尺寸
  • border-image-width:宽度
  • border-image-repeat: (图片的拉伸方式) stretch/round/repeat


注意：如果设置repeat不设置slice相框会出现在四周

transition 过渡

  • transition-property: width,height;（要变化的属性）
  • transition-duration: 2s;（过渡的时间）
  • transition-delay: 2s;（延迟的时间）
  • transition-timing-function: ease / easein / easeout / easeinout / linear(匀速) /cubic-bezier(贝塞尔曲线)

    transition 复合写法

  • 属性 完成过渡 延迟时间 过度方式

步骤动画 transition:all 7s steps(700)

说明:7秒之内走700步

transform

transform :translate（移动）

  • transform: translateX(-100px) translateY(100px); x轴 是横的 正方向是 → (负数表示反方向←) y轴是竖的 正方向时候 ↓
  • transform: translate(50px,50px); 表示X和Y轴

    transform: rotate(度数+deg) 表示旋转（默认沿着Z轴）

  • transform: rotate(36000deg); 沿着Z轴旋转
  • transform: rotateX(360deg) 沿着x轴旋转
  • transform: rotateY(360deg) 沿着Y轴旋转

    transform: scale() 缩放，默认是1

  • transform :scale(.5,.5);
  • transform :scale(1,2);

    transform :skew设置扭曲 ，一般不用

  • transform: skewX(-45deg);

    transform连写

  • transform: translateX(200px) rotate(180deg) scale(0.1, 0.1);
