常用函数

  • count(); 数组的长度

  • file_get_contents(); 读取文件里面的内容

  • .in_array(); 判断数组里面是否存在某个元素

    参数1:元素

    参数2:数组

  • array_key_exists(); 数组中是否存在key

  • move_uploaded_file()移动上传文件
  • 1：必须表单提交
    2：必须post 提交
    3：必须input type file
    4：表单必须设置一个属性 enctype=“multipart/form-data”
    服务端：
    $_FILES 去接收数据，使用move_uploaded_File 去移动上传的文件到对应的目录位置
    参数1：文件的临时存储位置
    参数2：保存的位置
