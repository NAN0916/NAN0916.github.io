bs和cs，客户端和服务端

bs架构/cs架构和客户端/服务端

bs和cs

bs(browser Server client 浏览器 服务器)

  • eg: 天猫，淘宝，百合网，珍爱网，有缘网，

    cs 架构 client ，server 客户端（需要安装）

  • eg: qq，快播，迅雷，lol，dota

    区别

  • 性能：

  • 安全性：

客户端和服务端

客户端

  • 具有向服务器索取服务能力的终端(手机，paid)
  • 常见的客户端软件：
    • 浏览器、QQ、迅雷、Foxmail等。

      服务端

  • 对外提供服务器的，一般的客户端都对应有服务端.
  • 服务器类型
    • 服务类型可分： 文件服务器、数据库服务器
    • 操作系统可分： Linux服务器、Windows服务器
    • 应用软件可分： Apache服务器、Nginx 服务器、IIS服务器、Tomcat服务器、Node服务器等。
  • 服务器软件有：
    • 文件服务器：Server-U、FileZilla、VsFTP等
    • 数据库服务器：Oracle【甲骨文】、MySQL、PostgreSQL、MSSQL,sqlServer等；
    • 邮件服务器：Postfix、Sendmail等；
    • web 服务器：Apache、Nginx、IIS、Tomcat、NodeJS等；
